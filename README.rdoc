== README

Jukebox is a fun music player for house parties, social gatherings, etc making it possible for all of your guests to be djs for the night. Just sign up to create your own jukebox. Then click on the "Request" drop down where you can enter in each of your guests phone numbers to send them a text message with a link to submit their song requests. Also, if you are a member you can see other user's jukebox and just simply select their name from the list in the "Request" menu to submit song requests to their jukebox.
